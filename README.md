## Requirements
This coursework uses only:
- `numpy`
- `scipy`
- `matplotlib`
- `torch`

Please make sure these libraries are installed in your Python environment.

## Repository structure
- `src/lqr_solver.py`: solver for Exercise 1.1
- `src/monte_carlo.py`: Monte Carlo validation for Exercise 1.2
- `src/networks.py`: supervised learning for Exercises 2.1 and 2.2
- `src/dgm.py`: Deep Galerkin Method for Exercise 3.1
- `src/policy_iteration.py`: policy iteration with DGM for Exercise 4.1
- `figures/`: saved figures generated by the code

## Reproducing the current outputs

Before running the scripts below, make sure the folder `figures/` exists in the repository root.

### Exercise 1.1
Run:
```bash
python src/lqr_solver.py
```

This performs a basic sanity check for:

the Riccati ODE solver
the value function
the optimal control

In particular, the script checks that the terminal condition is handled correctly.
